Minimalistic Finger ECG Board
Thar Htet Nyan*

A compact ECG sensing board designed to capture heart signals using only two thumbs and one toe for grounding. Built with an AD620 instrumentation amplifier, the system supports real-time signal display via oscilloscope. Sized like an iPhone, it offers a portable and minimalistic approach to biosignal monitoring.
